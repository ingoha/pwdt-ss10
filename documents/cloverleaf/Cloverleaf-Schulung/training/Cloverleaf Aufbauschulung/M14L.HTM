Integrator - Level II Training

Module 14L: Lab: Level I refresher

The goal of this lab is to provide you with a refresher of the knowledge acquired during the Level I training. The environment developed during this lab will be used as a basis for later labs as well.

This lab is based on the Java Kitchen Management Application. The specification of the JKMA interface as well as some guidelines for its implementation are provided to you in appendix C.

The tasks that are to be performed during this lab are described in the guidelines mentioned above. You have the choice of following one of the 3 available tracks:

  • HL7 2.x format (paragraph 2.1, 2.2.1, 2.3, 2.3.1 and 2.4)
  • EDIFACT format (paragraph 2.1, 2.2.2, 2.3, 2.3.2 and 2.4). This track should only be followed by those that are familiar with the EDIFACT message standard.
  • XML format (paragraph 2.1, 2.2.3, 2.3, 2.3.3 and 2.4)
Perform the tasks described in the guideline (see the above description for paragraph numbers). You will have to create a connection between a third-party system (using an HL7 or EDIFACT variant or a XML package) and the JKMA application (which uses an FRL based format).
  • Message layout definitions have to be created and tested;
  • A translation has to be created and tested;
  • The connection has to be created using the network configurator.
