SMAT

Das SMAT Tool kann eingesetzt werden, um die Inbound bzw. die Outbound Nachrichten oder auch beide, aus Sicherheitsgründen, zusätzlich zu sichern.

·       Inbound Nachrichten können auf diese Weise erneut an alle angeschlossenen Systeme versendet werden.

·       Outbound Nachrichten können auf diesem Weg erneut an angeschlossene Systeme versendet werden, sollten es aus irgendwelchen Gründen nötig sein (Desaster Recovery).

Das SMAT Tool speichert zudem für jede Nachricht das Datum und die Uhrzeit des Empfanges bzw. des erfolgreichen Versendens. Diese Information kann bei Unstimmigkeiten mit unter sehr wichtig sein.

 

Die Aktivierung der SMAT Funktion erfolgt bei jedem Thread individuell und wird bei der Thread Konfiguration auf dem Reiter Thread eingestellt. Neben der eigentlichen Aktivierung muss ein Dateiname für die SMAT Datei eingegeben werden.

 

 

Wird ausschließlich ein Dateiname eingegeben, speichert Cloverleaf die SMAT Dateien in das „Prozess Working Directory“. Die SMAT Dateien bestehen aus:

o      Index File (.idx)
In dieser Datei werden Informationen über die Nachricht gespeichert.

o      Message File (.msg)
In dieser Datei werden die Nachrichten selber gespeichert.

o      Format File (.ecd)
In dieser Datei wird das Datensatzformat gespeichert.

Die Menüleiste

Die Menüleiste enthält neben den Standardmenüs die optionalen SMAT bezogenen Menüs Selected und UnSelected. Die Standard Menüs File und View sind um einige Funktionen erweitert bzw. geändert.

 

File - Menü

Menüeintrag

Funktion

Load

Öffnet ein Fenster mit Dateibrowser zur Wahl einer SMAT Datei.

Close

Schließt das SMAT Fenster.

 

View - Menü

Menüeintrag

Funktion

Create New View …

Öffnet das Fenster Selected Messages (siehe unten).

Add All Messages To View

Alle Nachrichten der geöffneten SMAT Datei werden in die View geladen.

Add Specified Message To View …

Öffnet das Fenster Selected Message zur Angabe des Filters.

Remove All Messages From View

Entfernt alle Nachrichten aus der View.

Remove Specified Messages From View …

Öffnet das Fenster Selected Message zur Angabe des Filters.

Selected All Messages In View

Wählt (markiert) alle Nachrichten in der View.

Deselect All Message In View

Setzt die Liste der selektierten Nachrichten zurück.

 

Selected - Menü

Menüeintrag

Funktion

Resend…

Öffnet das Fenster Resend Messages (siehe unten) zum erneuten Versenden der gewählten Nachrichten.

Delete…

Löscht die gewählten Nachrichten aus der SMAT Datei.

Add to View

Reduziert die Nachrichten in der View ausschließlich auf die selektierten Nachrichten.

Clear Selection

Setzt die Liste der selektierten Nachrichten zurück.

Der Bildschirm des SMAT Tools teilt sich in folgende Bereiche auf:

Bereich

Beschreibung

Current File

Der Bereich Current File hält folgende Informationen zur SMAT Datei bereit:

File Name

Name und Pfad der aktuellen SMAT Datei.

Total Messages

Gesamtanzahl der Nachrichten in der SMAT Datei.

Messages in View

Anzahl der Nachrichten in der aktuellen View.

Selected Messages

Anzahl der gewählten (markiert) Nachrichten.

Current Message

Der Bereich Current Message hält folgende Informationen zur den angezeigten Nachrichten bereit:

Current Message ID

Anzeige der aktuellen Message ID. Diese setzt sich aus den 3 Parts Domain, Hub und Site zusammen. Domain und Hub sind 0. Die Site ist die fortlaufende Nachrichtennummer innerhalb der Site.

Sending Date

Anzeige des Datums und der Zeit wann diese Nachricht empfangen bzw. gesendet wurde.

Sending Host

Anzeige des Senders dieser Nachricht.

Selected Encode Scheme

Auswahl der Zeichenkodierung. Pulldown Feld zur Auswahl der gegebenen Möglichkeiten. „ASCII no encoding is performed“ ist Default.

Seleceted Encode Format

Auswahlmöglichkeit aller in Cloverleaf angelegten Formate. Sobald hier ein Format gewählt wurde erscheint der Bereich „Format Option“.

Format Option

In Abhängigkeit der Auswahl des Feldes Seleceted Encode Format werden die unterschiedliche Format Optionen angeboten. Je höher der „Detail Level“ eingestellt ist, desto granularer ist die Anzeige in der View (analog dem Testingtool).

Hier eingegebene Formatoptionen werden in der .ecd Datei gespeichert. Beim nächsten Öffnen derselben SMAT Datei, werden die Format Optionen vorbelegt.

View

Anzeige einer Nachricht oder Datei.

Werden Nachrichten hintereinander empfangen, so werden die Nachrichten auch einzeln angezeigt.

Empfangene Semaphor Dateien werden ebenfalls angezeigt.

Enthält eine Datei mehrere Datensätze (z. B. NTISH), so wird die Datei als Ganzes angezeigt.

Outbound Nachrichten werden immer sequenziell angezeigt, da Cloverleaf die Nachrichten sequenziell ausliefert.

Funktionsweise

Während der Thread in eine SMAT Datei schreibt, kann das SMAT Tool die Datei nicht öffnen. Entweder muss der Thread gestoppt werden oder die SMAT Datei kann im Network Monitor gecyceld werden. Achtung, wird der Thread wieder gestartet, bevor das SMAT Tool die Datei geschlossen hat, so kann der Thread die neuen Datensätze nicht in die SMAT Datei sichern (Datenverlust).

 

·       Zum Arbeiten mit Nachrichten aus der SMAT Datei muss diese über den Befehl File / Load geöffnet werden.

·       Anschließen sollten die Nachrichten in die View mit dem Befehl View / Add All Messages To View geladen werden.

·       Über das Fenster Select Message  (View / Create New View…) können Filter aktiviert werden. Die Aktivierung des Filters erfolgt über das linke Pulldown Feld vor jeder Filtermöglichkeit.

Don’t Care                   ohne diesen Filter

Include Those              mit diesem Kriterium

Not Include Those       alle außer diesem Kriterium

 

Folgende Filter sind möglich:

 

Sending Date From:             Einschränkung über einen Zeitraum.
Amerikanisches Datums-und Zeitformat in Form von M/dd/yy H:mm:ss (from 02/19/07 0:00:00 to 02/30/07 0:00:00)

Message ID From:                Eingabe eines Bereiches von Message IDs.

Sending Host:                        Beschränkung auf die Nachrichten eines bestimmten Host.

Regular Expression:             Angabe einer beliebigen Zeichenfolge, case sensitiv.

 

·       Ist die Auswahl getroffen können die gewählten Nachrichten über den Befehl Selected / Resend entweder direkt an einen Thread gesendet oder in eine Datei gespeichert werden.

 

 

·       Nachrichten die in eine Datei gespeichert werden, können über den Network Monitor direkt, über einen Thread, in das System eingespielt werden.

 

 

Hinweis

Wenn Sie mit Bedacht den Namen der SMAT Datei wählen, können Sie leicht ableiten,
in welchen Thread Sie die selektierten Datensätze wieder senden müssen.

his1_ib_smat.idx                   Thread         =>     his_1
Context        =>     Inbound

Datensätze sind generell vom Typ Data (default).
