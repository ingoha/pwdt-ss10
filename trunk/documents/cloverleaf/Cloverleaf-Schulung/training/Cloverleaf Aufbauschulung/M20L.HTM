Module 20L: Lab: msg and msgmeta

1. Message handling Procedures
Create a new message with the content "red and green". Display the entire contents of the message and display the character range 0 to 2. Insert the string ", orange" at position 3 and dump the message. Finally, destroy the message.

2. Metadata Procedures
Create a new message of type data and class engine and dump all metadata associated with the message. Query and display the value of the MID metadata field and change the value of the USERDATA metadata field to "{filteroptions {A B}}". Write a procedure (don't use the msgdump command) that displays all Read/Write metadata fields using the layout FIELD = VALUE. Finally, destroy the message.

3. Metadata Flags
Create a new message and display the value of the FLAGS metadata field. Display the value of the FLAGS metadata field after changing the value of the proto_nak and last_in_group flags to TRUE. Change the value of the last_in_group flag to false and display the flags metadata again. Finally, destroy the message.
