HRL Datensatz

Der HRL (Hierarchical-Record Layout) Datensatz besteht aus einer der folgenden Kombinationen von FRL und / oder VRL Datensätzen innerhalb eines Datensatzes:

·         Gruppen von FRL und / oder VRL Datensätze

·         Sich wiederholende Datensatzstrukturen

·         Optionale Datensatzstrukturen

·         Verschachtelte Datensatzstrukturen

 

Die Konfigurationsdateien des HRL Configurators werden im Verzeichnis $HCISITEDIR/formats/ abgespeichert.

 

Die Formatdefinition einer HRL Struktur besteht aus einer Reihe von Schlüsseln (Segmenten) mit zugehörigen Werten (keyed list entries).

o        Jeder Schlüssel, bezogen auf ein Segment, hat einen eindeutigen Typ.

o        Der Typ ergibt sich aus einem oder mehreren logischen Datensatzstrukturen, die entweder eine FRL, VRL oder HRL Definition enthält.

o        Ist der Typ ein FRL oder VRL, dann ist die Segment-Feld Definition in den angegebenen FRL bzw. VRL Dateien enthalten.

o        Ist der Typ ein HRL, dann ist die Segment Definition in einer weiteren HRL Datei hinterlegt.

Wiederholungen und Bedingungen

Wiederholungs Optionen / Repeat Options

Folgende Wiederholungsrichtlinien sind möglich:

 

Option

Beschreibung

No Repeat

Keine Wiederholung.

Repeats

Anzahl der Wiederholungen bezogen auf das markierte HRL Segment. Als Eingabe wird ein numerischer Wert erwartet.

Repeat Block

Wiederholung bezogen auf eine Folge von HRL Segmenten. Das Start und Ende Segment wird als Eingabe erwartet.

Repeat While

Entspricht einer While-Schleife. Die Wiederholung erfolgt solange, wie ein bestimmtes Feld (Field) einen bestimmten Inhalt (Value) aufweist. Das Feld ist ein FRL oder VRL Feld der aktuellen HRL Definition.

Beispiel:        <FRL Segmentname>.<Feldname>

Um auf das Feld PatientenID im Segment PAT zu referenzieren wäre die Syntax PAT.PatientenID.

 

Bedingungen / Condition Options

Folgende Bedingungen sind möglich:

 

Condition

Beschreibung

No Condition

Keine Bedingung.

Condition

Ein HRL Segment wird ignoriert, es sei denn dass ein bestimmtes Feld (Field) den definierten Inhalt (Value) enthält.

Das Feld ist ein FRL oder VRL Feld aus einem vorausgehenden HRL Segment.
