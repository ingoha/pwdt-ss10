6.5 Erstellung der HL7 Version 2.2 Variante var_1

 

Ø        Erstellen Sie eine HL7 Variante basierend auf dem HL7 Version 2.2 Standard. Speichern Sie diese als var1 ab.

Ø        Erstellen Sie ein neues Feld mit folgender Spezifikation:

o        Feldnummer:  90001

o        Name:  Allergies

o        Typ:      String (ST)

o        Lenght:            25 bytes

Ø        Erstellen Sie ein das Segment Z01 zur Übermittlung der Allergiedaten

o        Fügen Sie das Feld 90001 ans Ende des Segmentes an

o        Deklarieren Sie das Feld 90001 als Optional

o        Das Feld 90001 kann bis zu. 5 mal wiederholbar sein

o        Lenght:            25 bytes

Ø        Ändern Sie die ADT_A03 und ADT_A10 Nachricht

o        Fügen Sie Segment Z01 jeweils ans Ende der Nachrichten an

o        Deklarieren Sie das Z01 Segment als Optional
