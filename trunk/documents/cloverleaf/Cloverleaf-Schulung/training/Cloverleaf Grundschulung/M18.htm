Remote Commands

Das Tool Remote Commands erlaubt die Remote-Steuerung von Prozessen ohne eine Kommando - Shell.

·       Befehle sind per Mausklick abrufbar.

·       Benötigte Parameter werden ggf. angezeigt.

·       Vollständige Befehle können direkt abgesetzt werden.

 

 

Die Menüleiste

Die Menüleiste enthält neben den Standardmenüs das optionale Remote Commands bezogene Menü Process.

 

Process - Menü

Menüeintrag

Funktion

Erase output before each run

Ist diese Option aktiviert, wird die Anzeige im Bereich Process vor jeder neuen Ausführung eines Befehls bereinigt. Standardmäßig ist diese Option aktiviert.

Clear Output

Bereinigt die Anzeige im Bereich Process.

Save Output To File

Ermöglicht die Speicherung der Ausgabe in eine Datei. Der Dateibrowser wird geöffnet.

End Process

Beendet den aktuellen, in Ausführung befindenden Befehl.

 

 

Funktionsweise

Das Fenster Remote Commands unterteilt sich in die Bereiche Command und Process.

 

Command Pane

Feld / Button

Funktion

Command

Enthält den Befehl der ausgeführt werden soll.

Clear

Bereinigt die Command Anzeige.

List Commands

Öffnet das Fenster zur Selektion eines Befehls (siehe unten).

History

Öffnet das Fenster Command History. Hier werden die zuletzt ausgeführten Befehle gelistet.

Execute

Startet die Ausführung des Befehls in der Anzeige Command.

 

 

 

Process Pane

Feld / Button

Funktion

Input + Send

Benötigen Befehle zur Laufzeit eine Eingabe, z. B. die Initialisierung der Datenbank, erfolgt diese Texteingabe über das Feld Input. Mit dem Button Send wird die Eingabe abgeschickt.

End Process

Befehlsausführung wird abgebrochen.

Output List Box

Anzeige des ausgeführten Befehls und dessen Rückmeldung.

Clear Output

Bereinigt das Fenster Output List Box.

Save Output To File

Ermöglicht die Speicherung der Ausgabe in eine Datei. Der Dateibrowser wird geöffnet.
